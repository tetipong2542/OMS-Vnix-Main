When to Use This

Use this sync function when:

  • You edited data directly in Turso Dashboard (via Turso CLI or Web UI)
  • Changes made elsewhere aren't showing up in the web app
  • You want to ensure you have the latest data from the cloud

Note: Normal write operations (editing data through this web app) automatically sync to Turso - you don't need manual sync for those.

Manual Sync

This will download the latest data from all three Turso databases (Data, Price, Supplier) to the local embedded replica files. The process may take 10-30 seconds depending on database size.
